# AI Thumbnail Generator

An AI-powered Thumbnail Generator that enables users to create professional, high-quality thumbnails for YouTube videos, social media content, blogs, and marketing campaigns using Generative AI.

The application leverages Google's Generative AI capabilities to transform user prompts into visually appealing thumbnails while offering customization options such as style selection, aspect ratios, color schemes, and text overlays.

---

## 🚀 Project Overview

AI Thumbnail Generator is a full-stack web application designed to automate thumbnail creation through Artificial Intelligence. Users can describe their desired thumbnail, select visual preferences, and generate unique AI-powered designs within seconds.

The platform provides authentication, thumbnail history management, cloud image storage, and a responsive user experience, making it suitable for content creators, marketers, designers, and social media professionals.
